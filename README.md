## ShipWise Quoting Tool
Streamlit app to give to clients so they can rate their own quotes (usually international shipments)

# Table of Contents
-[Usage](#usage)


# Usage
1. Clone git repository
```bash
git clone https://github.com/3PLWinner/ShipWise-Rate-App.git
```

2. Create and Activate virtual environment
```bash
python -m venv env
env/Scripts/activate
```
